🖥️ Portfólio — Luiz Eduardo Garcez

Portfólio pessoal desenvolvido com React 19 e Vite, apresentando projetos, habilidades técnicas e informações de contato. Design dark com identidade visual elétrica, fundo animado com Canvas API e animações fluidas por toda a interface.

🌐 Sobre

Site single-page que apresenta Luiz Eduardo Garcez como Desenvolvedor Fullstack + Infra, baseado em Niterói, RJ. O portfólio reúne projetos reais entregues, stack técnica e canais de contato.

🛠️ Tecnologias Utilizadas

- React 19 — componentes funcionais com hooks
- Vite 8 — bundler e servidor de desenvolvimento
- JavaScript (JSX) — sem TypeScript, tudo em `.jsx`
- CSS-in-JS — estilos inline com variáveis CSS globais
- Canvas API — fundo animado com partículas, grid e scan line
- Google Fonts — Syne, DM Mono e Inter

📁 Estrutura do Projeto

portfolio-luiz/
├── index.html              # Entry point HTML
├── main.jsx                # Monta o React no DOM

├── GlobalBackground.jsx    # Canvas de fundo animado
├── InteractiveCursor.jsx   # Cursor personalizado
├── style.css               # Estilos globais e utilitários
├── vite.config.js          # Configuração do Vite
├── package.json
└── public/
    └── videos/             # Vídeos demo dos projetos (MP4)
📚 Seções do Site

| Seção | Descrição |
|---|---|
| Hero | Apresentação com nome, título, stack e estatísticas animadas |
| Skills | Bento grid com Frontend, Backend, Infra, Ferramentas, Formação e Idiomas |
| Projetos | Cards com vídeo demo ao hover, tags de tecnologia e links |
| Sobre | Trajetória pessoal e profissional |
| Contato | Links para GitHub, LinkedIn, e-mail e telefone |

🗂️ Projetos Exibidos

- Gestão Clínica — Espaço CuidadosaMente — sistema fullstack com React, Supabase e PostgreSQL
- Study Math — plataforma educacional responsiva
- Site Institucional — CuidadosaMente — site institucional com identidade visual da clínica
- DOGO-WORLD — jogo 2D em JavaScript puro com Canvas
- Calculadora de IP — ferramenta de cálculo de sub-redes para profissionais de infra

✨ Funcionalidades

- Fundo animado com partículas interativas, grid e scan line via Canvas API
- Cursor personalizado com efeito de seguimento suave
- Contadores animados nas estatísticas do Hero (easing cubic)
- Cards de projeto com reprodução de vídeo demo ao hover
- Animações de entrada (Reveal) com IntersectionObserver
- Navegação com menu mobile responsivo
- Layout bento grid adaptativo para Skills
- Totalmente responsivo — breakpoints em 900px, 560px e 375px
